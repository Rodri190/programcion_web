
# Juego de Tablero 10 x 10

## Descripción

Juego de tablero donde el usuario compite contra la computadora por turnos, expandiendo su color en un tablero de 10 x 10.

## Funcionamiento

* El jugador es color rojo y el computador verde
* Cada turno pinta la casilla seleccionada y sus adyacentes (incluyendo diagonales)
* La máquina juega automáticamente en una posición aleatoria
* El juego termina cuando el tablero está completo

## Reglas

* No se pueden seleccionar casillas ocupadas
* Gana quien tenga más casillas al final

## Características

* Interacción con eventos de clic
* Sistema de turnos
* Cálculo automático del ganador
* Persistencia con localStorage (estado)
## Tecnologías

* HTML
* CSS
* JavaScript

## Ejecución

Abrir `index.html` en el navegador.
